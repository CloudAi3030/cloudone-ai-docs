Welcome to the cloudone-ai.com API documentation. Our API allows you to programmatically interact with the cloudone-ai.com platform, enabling you to create and manage leads, schedule calls, and more.

## Base URL

All API requests should be made to the following base URL:

```
https://app.cloudone-ai.com/api
```
